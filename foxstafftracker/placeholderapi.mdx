---
title: "FoxStaffTracker PlaceholderAPI Placeholder'ları"
sidebarTitle: "PlaceholderAPI"
description: "FoxStaffTracker'ın PlaceholderAPI ile sunduğu %foxstaff_...% placeholder'larının tam listesi: süre, mesaj, rütbe, aktiflik ve AFK durumları."
---

PlaceholderAPI kuruluysa `%foxstaff_...%` placeholder'ları kullanılabilir. **Sadece cache'ten** okunur (çevrimiçi yetkililer için hızlıdır; çevrimdışı için varsayılan değer döner).

| Placeholder | Örnek Çıktı | Açıklama |
| --- | --- | --- |
| `%foxstaff_playtime_daily%` | `2s 15dk` | Günlük süre (biçimli) |
| `%foxstaff_playtime_weekly%` | `12s 30dk` | Haftalık süre |
| `%foxstaff_playtime_monthly%` | `40s 0dk` | Aylık süre |
| `%foxstaff_playtime_total%` | `310s 45dk` | Toplam süre |
| `%foxstaff_minutes_daily%` | `135` | Günlük süre (ham dakika) |
| `%foxstaff_minutes_weekly%` | `750` | Haftalık (ham dakika) |
| `%foxstaff_minutes_monthly%` | `2400` | Aylık (ham dakika) |
| `%foxstaff_minutes_total%` | `18645` | Toplam (ham dakika) |
| `%foxstaff_messages_daily%` | `42` | Günlük mesaj sayısı |
| `%foxstaff_messages_weekly%` | `310` | Haftalık mesaj sayısı |
| `%foxstaff_messages_monthly%` | `1200` | Aylık mesaj sayısı |
| `%foxstaff_messages_total%` | `9800` | Toplam mesaj sayısı |
| `%foxstaff_role%` | `admin` | Yetkilinin rütbesi |
| `%foxstaff_active%` | `true` / `false` | Şu an çevrimiçi (aktif) mi |
| `%foxstaff_afk%` | `true` / `false` | Şu an AFK mı |
| `%foxstaff_is_staff%` | `true` / `false` | Takip edilen yetkili mi |
