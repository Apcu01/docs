---

Eklenti ilk çalıştığında otomatik oluşturur (MySQL):

| Tablo | İçerik |
| --- | --- |
| `fox_staff_data` | Yetkili ana verisi: süre/mesaj sayacları, rol, aktiflik, son görülme |
| `fox_server_playtime` | Oyun modu (sunucu) bazlı günlük süre |
| `fox_playtime_history` | Günlük oynama süresi geçmişi (grafik için) |
| `fox_chat_logs` | Yetkili sohbet mesajları |
| `fox_command_logs` | Yetkili komut kayıtları |
| `fox_moderation_logs` | Moderasyon işlemleri (LiteBans) |
| `fox_activity_goals` | Role göre aktivite hedefleri |
| `fox_staff_reports` | Oyuncu şikayetleri |
