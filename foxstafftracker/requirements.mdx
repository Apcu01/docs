description: "FoxStaffTracker eklentisi ve web panelinin çalışması için gereken sunucu, Java, veritabanı ve opsiyonel eklenti gereksinimleri."
---

| Bileşen | Gereksinim |
| --- | --- |
| Sunucu yazılımı | **Paper** (veya uyumlu fork), `api-version 1.16+` |
| Java | **Java 21\+** |
| Veritabanı | **MySQL 5.7\+ / MariaDB** (panel ve çoklu sunucu için zorunlu) |
| Panel çalışma ortamı | **Node.js 20\+** |
| Opsiyonel eklentiler | LuckPerms, LiteBans, EssentialsX, PlaceholderAPI |

<Note>
  MySQL, web panel ve çoklu sunucu kurulumları için **zorunludur**. YAML depolama yalnızca tek sunucu için temel kullanım sağlar.
</Note>
